# DM — Telegram-бот личного ассистента

Персональный бот-компаньон на каждый день. Совмещает утреннюю сводку, гардероб, изучение языков, досуг, здоровье и свободный чат в одном месте.

---

## Команды

| Команда | Описание |
|---|---|
| `/start` | Главное меню |
| `/setup` | Настройки (город, язык, уведомления, параметры тела) |
| `/notes` | Закладки и любимое |
| `/plany` | Запустить утреннюю сводку вручную |
| `/weather [дни]` | Погода на 1–7 дней |
| `/setcity` | Сменить город |
| `/reload_wardrobe` | Перезагрузить шкаф из `wardrobe.json` |
| `/reload_content` | Перезагрузить списки фильмов/книг из `content.json` |
| `/reload_artists` | Перезагрузить список артистов из `artists.json` |

---

## Автоматические уведомления

| Время | Событие |
|---|---|
| 08:30 | ☀️ Утренняя сводка — погода, образ, факт дня, идея |
| 11:00 | 📚 Слово/грамматика дня (NL или EN) |
| 14:00 | 🫣 Дневная разгрузка — выгрузи тревоги текстом |
| 20:00 | 🥸 Вечерний разбор — что из тревог было реальным |

Каждое уведомление можно включить/выключить в `/setup → Уведомления`.

## Разделы меню

### ☀️ Мой день
- Утренняя сводка: погода, рекомендация образа, факт о городе, идея дня, цитата
- Дневник: запись мыслей на день
- Дневная разгрузка: список тревог, которые вечером разбираются на «факт» / «отпустить»
- Мотивационная фраза

### 👕 Гардероб
- Подбор лука по текущей погоде (тип одежды, температура, осадки)
- Просмотр всего шкафа по категориям
- Добавление вещей текстом или файлом `.txt`
- Оценка покупки — стоит ли брать (по стилю и параметрам пользователя)

### 🧠 Баланс
- Вопрос врачу — структурированный разбор симптомов или лекарств
- Мотивация для СДВГ — микро-техники, один шаг прямо сейчас
- Идея дня — мини-проект (дизайн, ИИ, AR, путешествия)
- Рецепты — завтрак / обед / ужин, можно по остаткам из холодильника

### 📚 Обучение
- Нидерландский и английский: перевод слов, грамматические упражнения (A/B), пословицы, темы для разбора
- Личный словарь: слова и фразы с переводом, редактирование
- Игра-угадайка: настраиваемый язык и сложность (easy / med / hard), подсказки, раскрыть ответ
- Уровень языка (A1–C2) — влияет на сложность заданий

### 🍿 Досуг
- Фильмы и сериалы: рекомендации, список к просмотру, добавление в любимые / чёрный список
- Книги: рекомендации, список к прочтению
- Музыка: рекомендации треков, лента любимых
- Концерты: поиск по домашней стране или выбранной европейской стране (Ticketmaster)
- Путешествия: подборка поездок по любимым странам, планирование маршрута

### ⚙️ Настройки (`/setup`)
- Уведомления — включить/выключить каждый тип
- Язык обучения — нидерландский или английский
- Уровень языков — A1 / A2 / B1 / B2 / C1 / C2
- Город — изменить через текст или геолокацию
- Параметры тела — рост, вес, размер одежды и обуви (используются для подбора лука)
- Стиль — минимализм / скандинавская эстетика / натуральные ткани
- Шкаф — просмотр, добавление, удаление вещей

## Закладки и любимое (`/notes`)

- **⭐ Временные закладки** — любой ответ бота можно сохранить кнопкой «⭐ В закладки». При удалении можно переместить в «Любимые» или «Чёрный список».
- **❤️ Любимые** — постоянные списки: страны, артисты, книги, шкаф, Лагом-практики.
- **📤 Экспорт** — выгрузка всех сохранений в текстовый файл.

---

## Свободный чат

Любое сообщение вне меню уходит в AI-чат. Бот помнит последние 10 сообщений. При упоминании симптомов предлагает перейти к «Вопросу врачу».

## Технический стек

- **Python** — `python-telegram-bot` (polling)
- **AI** — Gemini (основной), Anthropic Claude, Groq, OpenAI, OpenRouter (резерв)
- **Поиск/ранжирование** — ZeroEntropy (rerank)
- **Контент** — TMDB (фильмы), Ticketmaster (концерты)
- **Хранилище** — JSON-файлы (Railway Volumes / локально)
- **Часовой пояс** — `Europe/Amsterdam`

## Переменные окружения

| Переменная | Назначение |
|---|---|
| `TELEGRAM_TOKEN` | Токен бота |
| `GEMINI_API_KEY` | Основная LLM |
| `ANTHROPIC_API_KEY` | Claude (опционально) |
| `GROQ_API_KEY` | Groq (опционально) |
| `OPENAI_API_KEY` | OpenAI (опционально) |
| `OPENROUTER_API_KEY` | OpenRouter (опционально) |
| `OPENROUTER_MODEL` | Модель OpenRouter (по умолч. `meta-llama/llama-3.3-70b-instruct:free`) |
| `OPENAI_MODEL` | Модель OpenAI (по умолч. `gpt-4o-mini`) |
| `CHAT_ID` | Telegram chat ID для авто-уведомлений |
| `DATABASE_URL` | PostgreSQL (если используется) |
| `ZEROENTROPY_API_KEY` | ZeroEntropy rerank |
| `CLOUDFLARE_API_TOKEN` / `CLOUDFLARE_ACCOUNT_ID` | Cloudflare Workers AI |
| `TICKETMASTER_API_KEY` | Концерты |
| `TMDB_API_KEY` | Фильмы/сериалы |
