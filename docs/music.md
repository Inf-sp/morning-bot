# Музыка и концерты

## Назначение
Этот раздел помогает находить новую музыку под свой вкус и не пропускать концерты любимых исполнителей. В меню это разделено на два понятных входа: «🎧 Музыка для тебя» для персональной рекомендации и «🎤 Концерты» для поиска реальных выступлений.

## Как это работает
Сценарий «🎧 Музыка для тебя» начинается со списка любимых исполнителей, который указывает пользователь. Если список пуст, бот сначала попросит прислать несколько имён (через запятую или с новой строки). На основе этого списка бот подбирает одного нового исполнителя, максимально близкого по звучанию, добавляет короткое описание, пояснение «почему зайдёт», несколько треков, с которых стоит начать, и интересный факт. Для актуальности бот подглядывает свежие данные из сети, чтобы советовать реально существующие треки и альбомы, а не выдуманные.

Концерты ищутся через Ticketmaster — по каждому любимому исполнителю бот проверяет, не выступает ли он в ближайшие полгода в выбранной стране. Результаты по стране кэшируются на неделю, чтобы не делать один и тот же запрос повторно и не перегружать внешний сервис. Из результатов исключаются трибьют-шоу, кавер-концерты и похожие имитации — остаются только выступления самого артиста.

Отдельно есть еженедельная проверка: раз в неделю (по воскресеньям, вместе с утренней афишей на неделю) бот обновляет список концертов и сравнивает его с тем, что видел раньше. Если появился новый концерт кого-то из любимых исполнителей — присылает отдельное уведомление именно про это; если ничего нового не появилось, уведомление не приходит вообще (в отличие от общей еженедельной афиши, которая приходит всегда). При самом первом включении этой проверки бот просто запоминает текущие концерты как «уже видел» и ничего не присылает — иначе первый запуск показал бы всю афишу как «новое».

## Основные возможности
- «🎧 Музыка для тебя» — один новый исполнитель под вкус, с описанием звучания, объяснением сходства и рекомендованными треками.
- Кнопки под карточкой: «✨ Заменить» — широкая строка сверху, показывает другого исполнителя, отклонённый уходит в чёрный список; ниже, в два столбца — «❤️ В любимые» (явно добавляет исполнителя в «Мои музыканты», влияет на подбор) и «⭐️ Сохранить» (кладёт исполнителя в закладки «Сохранённое», не влияет на вкус).
- Экран «🚫 Скрытое» — отклонённые исполнители, с действием «Вернуть в рекомендации».
- Быстрый переход в настройки списка любимых музыкантов прямо из карточки.
- Поиск концертов любимых исполнителей в выбранной стране на ближайшие полгода, с ценой, местом и датой.
- Возможность сменить страну поиска концертов из списка популярных европейских стран.
- Отдельное уведомление о новых концертах любимых исполнителей — приходит только когда действительно появилось что-то новое.
- Еженедельная афиша (по воскресеньям) с концертами любимых артистов и ближайшими кинопремьерами.

## Настройки
- Список любимых исполнителей — можно пополнять вручную, а также автоматически, когда пользователь сохраняет понравившегося артиста из подборки.
- Список исполнителей, которые уже знакомы или не понравились — не будут предлагаться повторно.
- Страна для поиска концертов — выбирается из списка (Нидерланды, Бельгия, Германия, Франция, Великобритания, Испания, Италия, Австрия, Швейцария, Польша, Швеция, Дания, Португалия), по умолчанию берётся страна из общих настроек профиля.
- Уведомления о новых концертах любимых артистов — включаются автоматически вместе с использованием раздела «Концерты» и еженедельной афишей; отдельного переключателя «вкл/выкл» в коде нет, работа привязана к наличию списка любимых артистов и ключа доступа к внешнему сервису.

## Ограничения
- Сценарий «Музыка для тебя» и поиск концертов не заработают без заполненного списка любимых исполнителей — бот сначала попросит его прислать.
- Поиск концертов требует включённого доступа к внешнему сервису по продаже билетов; если он не настроен, бот честно сообщает, что функция недоступна, вместо того чтобы показывать пустой список.
- Результаты по одной и той же стране обновляются не чаще раза в неделю (используется недельный кэш) — сразу после смены страны первый запрос может занять время, а следующие несколько дней будут показывать одни и те же данные.
- Трибьют-концерты, кавер-шоу и похожие «имитации» исполнителя намеренно скрываются из результатов.
- Список стран для поиска концертов ограничен Европой — другие регионы не поддерживаются.
- Если внешний сервис долго не отвечает или превышен лимит запросов, часть исполнителей может временно не попасть в список концертов — бот не переспрашивает бесконечно, чтобы не заблокировать всю проверку.

## Что можно улучшить
- Дать пользователю возможность явно включать/выключать уведомления о новых концертах, а не только неявно через наличие списка артистов.
- Расширить список доступных стран для поиска концертов за пределы Европы.
- Показывать в подборе музыкантов ссылку для прослушивания трека прямо в карточке.
- Позволить отмечать «не интересно» не только для целого артиста, но и для конкретного города/концерта.
