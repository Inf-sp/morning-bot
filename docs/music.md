# Музыка и концерты

## Назначение
Этот раздел помогает находить новую музыку под свой вкус и не пропускать концерты любимых исполнителей. В меню это разделено на два понятных входа: «🎧 Музыка для тебя» для персональной рекомендации и «🎤 Концерты» для поиска реальных выступлений.

## Как это работает
Сценарий «🎧 Музыка для тебя» начинается со списка любимых исполнителей, который указывает пользователь. Если список пуст, бот сначала попросит прислать несколько имён (через запятую или с новой строки). На основе этого списка бот подбирает одного нового исполнителя, максимально близкого по звучанию, добавляет короткое описание, пояснение «почему зайдёт», несколько треков, с которых стоит начать, и интересный факт.

Изучаемый язык используется как сильный дополнительный сигнал. Например, при выбранном нидерландском бот в первую очередь ищет современного заметного нидерландоязычного исполнителя, который совпадает с музыкальным вкусом; ориентиром может быть `Eefje de Visser — De Parade`. Это не жёсткий фильтр: точность по звучанию важнее формального совпадения языка. Предпочтение отдаётся активным, популярным или признанным в своей сцене артистам с выразительной мелодичной музыкой, а не случайным малоизвестным именам. Для актуальности бот использует свежие данные из сети и просит модель указывать только реально существующие треки и альбомы.

Концерты ищутся через Ticketmaster — основной источник, но не единственный. Для артистов, которых Ticketmaster не нашёл, бот дополнительно раз в 7 дней проверяет официальный сайт исполнителя, сайты концертных площадок и другие билетные сервисы через веб-поиск, чтобы найти события с небольших площадок и у локальных промоутеров. За один запуск внешний поиск берёт не более пяти таких артистов и выполняет не больше двух запросов одновременно, поэтому не создаёт всплеск запросов и лимитов. Найденные страницы бот честно читает и извлекает из них дату, площадку, город, формат события и ссылки — событие добавляется только если основные данные прямо подтверждены текстом страницы, ничего не додумывается. Карточка отдельно сообщает, выступает артист с собственным концертом или на фестивале; для фестиваля показывает его название. Оба источника объединяются в один список без дублей (по исполнителю, дате, городу и площадке); если одно и то же событие нашлось в нескольких источниках, побеждает более надёжный — сайт исполнителя, затем сайт площадки, затем Ticketmaster, затем прочие билетные сервисы. Поиск идёт по стране пользователя и её ближайшим соседям. Результаты кэшируются: Ticketmaster-подборка по стране — на неделю, внешний поиск по каждому исполнителю — тоже на неделю, но глобально (общий кэш для всех пользователей с этим исполнителем). При добавлении или удалении любимого исполнителя сводная подборка сразу сбрасывается. Из результатов исключаются трибьют-шоу, кавер-концерты и похожие имитации — остаются только выступления самого артиста.

Отдельно есть еженедельная проверка: раз в неделю (по пятницам в 10:00, вместе с афишей недели — плановое уведомление «Куда сходить») бот обновляет список концертов и сравнивает его с тем, что видел раньше. Если появился новый концерт кого-то из любимых исполнителей — добавляет его в то же сообщение отдельным блоком; если ничего нового не появилось, показывается только афиша, без лишнего блока. При самом первом включении этой проверки бот просто запоминает текущие концерты как «уже видел» и ничего не присылает — иначе первый запуск показал бы всю афишу как «новое».

## Основные возможности
- Главный экран: `✨ Подобрать музыку`, `❤️ Мои артисты`, `💾 Послушать позже`, `🎚️ Предпочтения`, `⬅️ Назад`, `#️⃣ Главная`.
- «🎧 Музыка для тебя» — один новый исполнитель под вкус, с описанием звучания, объяснением сходства и рекомендованными треками.
- Замена и добавление исполнителя в любимые сразу показывают следующую карточку без промежуточного сообщения в чат.
- Выбранный язык обучения влияет на подбор: исполнители на этом языке получают приоритет, если одновременно подходят по музыкальному вкусу.
- Кнопки под карточкой: «✨ Заменить» — широкая строка сверху, показывает другого исполнителя, отклонённый уходит в чёрный список; ниже, в два столбца — «❤️ В любимые» (явно добавляет исполнителя в «Мои музыканты», влияет на подбор) и «💾 Сохранить». После сохранения бот сразу подбирает следующего исполнителя; повторное нажатие на старой карточке убирает сохранение без нового подбора.
- Отклонённые, знакомые и удалённые исполнители попадают в общую внутреннюю категорию «Не рекомендовать» и не повторяются; отдельных кнопок «Скрытое» и «Уже знаю» нет.
- Быстрый переход в настройки списка любимых музыкантов прямо из карточки.
- Поиск концертов любимых исполнителей в выбранной стране на ближайший год, с ценой, местом и датой; если событие найдено не через Ticketmaster, карточка показывает, откуда оно взято (сайт исполнителя, сайт площадки, билетный сервис).
- Возможность сменить страну поиска концертов из списка популярных европейских стран.
- Плановое уведомление «Куда сходить» (по пятницам в 10:00) — афиша недели (концерты любимых артистов + ближайшие кинопремьеры) и, если появилось что-то новое, новые концерты любимых исполнителей — одним сообщением.

## Настройки
- Список любимых исполнителей — можно пополнять вручную, а также автоматически, когда пользователь сохраняет понравившегося артиста из подборки.
- Список исполнителей, которые уже знакомы или не понравились — не будут предлагаться повторно.
- Страна для поиска концертов — выбирается из списка (Нидерланды, Бельгия, Германия, Франция, Великобритания, Испания, Италия, Австрия, Швейцария, Польша, Швеция, Дания, Португалия), по умолчанию берётся страна из общих настроек профиля.
- Новые концерты любимых артистов показываются автоматически внутри «Куда сходить»; отдельного переключателя «вкл/выкл» в коде нет, работа привязана к наличию списка любимых артистов и ключа доступа к внешнему сервису.

## Ограничения
- Сценарий «Музыка для тебя» и поиск концертов не заработают без заполненного списка любимых исполнителей — бот сначала попросит его прислать.
- Поиск концертов требует включённого доступа к внешнему сервису по продаже билетов; если он не настроен, бот честно сообщает, что функция недоступна, вместо того чтобы показывать пустой список.
- Результаты по одной и той же стране обновляются не чаще раза в неделю (используется недельный кэш) — сразу после смены страны первый запрос может занять время, а следующие несколько дней будут показывать одни и те же данные.
- Трибьют-концерты, кавер-шоу и похожие «имитации» исполнителя намеренно скрываются из результатов.
- Список стран для поиска концертов ограничен Европой — другие регионы не поддерживаются.
- Если внешний сервис долго не отвечает или превышен лимит запросов, часть исполнителей может временно не попасть в список концертов — бот не переспрашивает бесконечно, чтобы не заблокировать всю проверку.
- Внешний поиск (не Ticketmaster) добавляет событие только если дата, исполнитель и площадка прямо подтверждены найденной страницей — если подтверждения нет, событие просто не попадает в список, без догадок.
- "Соседние регионы" для внешнего поиска — заранее заданный список географических соседей внутри той же группы европейских стран, что поддерживает поиск концертов, а не полная карта всех границ мира.

## Что можно улучшить
- Дать пользователю возможность явно включать/выключать уведомления о новых концертах, а не только неявно через наличие списка артистов.
- Расширить список доступных стран для поиска концертов за пределы Европы.
- Показывать в подборе музыкантов ссылку для прослушивания трека прямо в карточке.
- Позволить отмечать «не интересно» не только для целого артиста, но и для конкретного города/концерта.
