# ✈️ Поездки

## Назначение
Раздел предлагает одну практичную поездку на сегодня, помогает выбрать новое направление и хранит список уже посещённых стран.

## Как это работает
Главный экран показывает короткую идею маршрута из выбранного города: направление через стрелку `→`, до трёх пунктов маршрута, реальный прогресс и один полезный совет. Основной транспорт берётся из настроек пользователя. При каждом явном открытии раздела бот старается предложить новое направление; если генерация недоступна, показывает безопасный готовый маршрут для выбранного транспорта.

«🗺️ Мои страны» использует существующий список посещённых стран. Старые записи автоматически и без сброса переводятся на стабильные коды стран. Страны сортируются по алфавиту, показываются по две в ряд и по десять на странице.

При первом открытии конкретной страны бот собирает компактную карточку и сохраняет её в общем кэше по коду страны. Следующие открытия этой страны у любого пользователя читают готовую карточку из базы без AI-запроса. Удаление убирает только страну из личного списка и не удаляет общую карточку.

## Основные возможности
- Получить актуальную идею поездки из своего города
- Подобрать новую страну и собрать подробный план
- Открыть посещённые страны с пагинацией и компактными карточками
- Добавить или удалить посещённую страну без потери общего справочного кэша
- Сохранить готовый план поездки

Карточка посещённой страны показывает описание, чем она запоминается, языки на русском, валюту, главный нюанс и один факт.

## Настройки
Кнопка «*️⃣ Выбрать транспорт» позволяет отметить один или несколько вариантов:
- велосипед
- автобус
- поезд
- самолёт
- паром

Выбранные варианты влияют на идею дня, заголовок и план маршрута. Бот отдаёт приоритет выбранному транспорту и использует резерв только когда нормальный маршрут иначе невозможен.

## Ограничения
- Свежая идея и первая карточка страны зависят от доступности AI и внешних справочных данных
- Для старого названия, которого нет во встроенном справочнике, код страны уточняется при доступности сервиса стран
- Время в пути и бюджет остаются ориентирами; расписание нужно проверить перед выходом

## Что можно улучшить
