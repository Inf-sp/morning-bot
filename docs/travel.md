# ✈️ Поездки

## Назначение
Раздел предлагает одну практичную поездку на сегодня, помогает выбрать новое направление и хранит список уже посещённых стран.

## Как это работает
Главный экран показывает короткую идею маршрута из выбранного города: направление через стрелку `→`, до трёх пунктов маршрута, строку с жирной подписью «Прогресс:» и один полезный совет. Количество посещённых берётся напрямую из актуального списка «🧳 Мои страны»; сохранённые рекомендации в этот счётчик не входят. Основной транспорт берётся из настроек пользователя. Идея кэшируется на сутки и остаётся одинаковой при повторных открытиях. После смены города или транспорта она пересобирается сразу. Если генерация недоступна, бот сохраняет безопасный готовый маршрут для выбранного транспорта.

«🧳 Мои страны» использует существующий список посещённых стран. Старые записи автоматически и без сброса переводятся на стабильные ISO-коды стран. Технические идентификаторы, HEX-значения и записи без настоящего названия пользователю не показываются. Страны сортируются по названию, показываются по две в ряд и по десять на странице. Номер страницы находится только в кнопках пагинации.

Пагинация использует кнопки `◀️` и `▶️` и переключает страницы по кругу: после последней открывается первая, а перед первой — последняя.

«✨ Подобрать страну» сразу показывает короткую редакторскую карточку с одной горизонтальной туристической фотографией: характер поездки, одно конкретное совпадение со вкусами, три разноплановых места, сезон с причиной, бюджет с причиной, полезные языки и практичная LGBTQ+-оценка. Для персонализации выбираются только один-два сильных интереса, а самолёт, паром и велосипед не становятся аргументом сами по себе. Блок случайного факта не показывается. Промежуточного превью и кнопки «Показать подробности» нет. Кнопка «✨ Заменить страну» сразу подбирает другое направление. Бот ищет природный пейзаж, архитектуру или узнаваемое место через Pexels, отбрасывает портреты, людей, еду, карты, флаги и текстовые изображения, затем выбирает один лучший результат. Фото сохраняется вместе с карточкой; имя автора и источник не показываются. Если Pexels недоступен, используется Unsplash; если фото получить не удалось, карточка остаётся рабочей в текстовом виде.

Флаг рекомендации определяется по ISO-коду страны через справочник, а не берётся как произвольный эмодзи из AI-ответа. Поэтому декоративные символы вроде `🌟` перед названием не показываются.

Посещённые страны из «Моих стран» запрещены для новых рекомендаций. Проверка выполняется по ISO-коду после генерации, поэтому другое написание или язык названия не обходят запрет. Если модель всё же вернула посещённую страну, её название добавляется в следующую попытку: prompt меняется и не получает прежний ответ из AI-кэша. Кнопка `💾 Сохранить` добавляет рекомендацию в «Сохранённое → Мои поездки» и не помечает страну как посещённую.

Языки и готовые практические профили страны берутся из проверяемого справочника; модель пишет только редакторские строки по уже полученным данным. LGBTQ+-оценка никогда не берётся из модели: при отсутствии свежего проверенного профиля карточка честно сообщает об ограничении. Бюджет оценивается словами «низкий», «средний» или «высокий» с одной главной причиной стоимости; произвольные числовые суммы не генерируются.

При первом открытии конкретной страны бот собирает компактную карточку и сохраняет её в общем кэше по коду страны. Следующие открытия этой страны у любого пользователя читают готовую карточку из базы без AI-запроса. Удаление убирает только страну из личного списка и не удаляет общую карточку.

## Основные возможности
- Получить актуальную идею поездки из своего города
- Подобрать новую страну и сразу получить полную карточку
- Получить одну визуальную обложку рекомендованной страны
- Открыть посещённые страны с пагинацией и компактными карточками
- Добавить или удалить посещённую страну без потери общего справочного кэша
- Сохранить готовый план поездки

Карточка посещённой страны показывает описание, чем она запоминается, языки на русском, валюту, главный нюанс и один факт.

## Настройки
Кнопка «*️⃣ Выбрать транспорт» позволяет отметить один или несколько вариантов:
- велосипед
- автобус
- поезд
- самолёт
- паром

Выбранные варианты влияют на идею дня, заголовок и план маршрута. Бот отдаёт приоритет выбранному транспорту и использует резерв только когда нормальный маршрут иначе невозможен.

## Ограничения
- Свежая идея и первая карточка страны зависят от доступности AI и внешних справочных данных
- Для официального резерва фотографий требуется `UNSPLASH_ACCESS_KEY`; без него ошибка Pexels оставляет текстовую карточку
- Для старого названия, которого нет во встроенном справочнике, код страны уточняется при доступности сервиса стран
- Числовой бюджет не показывается без дат, длительности поездки и маршрута

## Что можно улучшить
- Учитывать текущую погоду и часы работы мест в маршруте дня
- Показывать время последнего подходящего обратного рейса
