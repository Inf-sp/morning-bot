# Тренажёр

## Назначение
Единый тренажёр учит все записи словаря одинаково — короткое слово и устойчивую фразу — без разделения на «тренажёр слов» и «тренажёр фраз». Идея: язык эффективнее учится через фразу и контекст, а не через голое слово, поэтому даже одно слово тренируется как короткая фраза с разбором и примером.

Слова и фразы добавляются в словарь прямо из чата или через кнопку «Добавить» в разделе словаря — см. `docs/dictionary.md`. После сохранения запись автоматически попадает в тренажёр нужного языка.

## Как это работает
Пользователь заходит в раздел языка и нажимает «Тренажёр». Бот берёт случайную запись из личного словаря (только записи с переводом) и собирает для неё учебную карточку и отдельный тест.

Карточка собирается программно из уже сохранённых при добавлении слова данных — термина, перевода, разбора и примера — без обращения к нейросети. Так тренажёр отвечает быстро и одинаково стабильно на каждый показ. Изредка (не каждый показ одного и того же слова) бот всё же генерирует новую карточку через ИИ для разнообразия примеров — этот результат не сохраняется обратно в словарь.

**Учебная карточка.** Сначала бот показывает термин целиком, естественный перевод, короткий разбор и новый пример:

```
🧩 Фраза-тренажёр

Hij is ziek door stress.
Перевод: Он болен из-за стресса.

💡 Разбор

ziek door iets — болеть или плохо себя чувствовать из-за причины.

Пример:
Hij blijft thuis door hoofdpijn.
→ Он остаётся дома из-за головной боли.

Дальше проверим это выражение на новом примере.
```

После заголовка `💡 Разбор` всегда идёт новая информация: значение конструкции, правило, особенность употребления или отличие от похожего выражения.

Под карточкой две кнопки:
- `🧩 Тест` — открывает тестовый пример по правилу из карточки.
- `✅ Выучил` — не открывает тест, не засчитывает правильный ответ и сразу показывает следующую учебную карточку.

**Тест.** После нажатия `🧩 Тест` бот случайно выбирает один из двух форматов, чтобы задания не приедались:

- **С вариантами.** Quiz poll в Telegram с новой фразой с пропуском и четырьмя вариантами: правильный целевой токен и три правдоподобных неправильных.

```
🧩 Проверь себя

Zij blijft thuis ____ de regen.
```

- **Да / нет.** Короткое утверждение — подставлено верное или одно из неверных слов — и две кнопки `✅ Да` / `❌ Нет`.

```
🤔 Верно или нет?

> Zij blijft thuis «de» de regen.
```

Оба формата используют один и тот же тест по одной и той же карточке — второй формат не требует отдельного запроса к ИИ.

**Если ответ правильный.** Бот показывает компактный результат без повторного объяснения, дальше доступна кнопка `Следующая`.

**Если ответ неправильный впервые.** Бот показывает короткое исправление и кнопки `Новый пример` / `Дальше`.

**Если пользователь ошибся повторно по тому же правилу.** Бот показывает `❌ Не закрепилось` и дополнительно кнопку `🔎 Разобрать` — короткий разбор только по явному нажатию.

## Связь учебной карточки и теста
Для карточек, собранных программно из сохранённого примера, эта проверка не нужна — данные уже сохранены при добавлении слова и не меняются на лету. Для редких карточек, которые всё же генерирует ИИ (для разнообразия), учебная фраза, правило и тест должны совпадать по смыслу — перед показом бот проверяет:
- русский перевод относится именно к учебной записи;
- паттерн реально присутствует в учебной записи;
- целевой токен используется в роли, которую описывает правило;
- пример естественный на выбранном языке;
- тестовая фраза проверяет то же правило и не является копией учебной;
- в одной карточке не смешаны разные значения одного слова.

## Основные возможности
- Единый формат для коротких слов и длинных фраз — без деления на типы.
- Карточка собирается программно из уже сохранённых данных слова — без AI-вызова на каждый показ.
- Учебная карточка с термином, переводом и коротким правилом.
- Тест на применение правила в новом контексте, а не на дословное восстановление.
- Два формата теста (с вариантами и да/нет), которые чередуются случайно.
- Компактный результат после правильного ответа, короткое исправление после ошибки.
- Кнопка `🔎 Разобрать` после повторной ошибки по тому же правилу.
- Кнопка `Новый пример`, которая генерирует новую тестовую фразу по тому же правилу.

## Настройки
- Выбор языка тренировки: нидерландский или английский — через соответствующий раздел меню.

## Ограничения
- Запись попадает в тренажёр только если в словаре у неё заполнен перевод; записи без перевода игнорируются.
- Если у записи нет сохранённого примера предложения (старый формат), бот один раз донасыщает её через ИИ и дальше использует сохранённое, как обычно.
- Редкие карточки для разнообразия (не каждый показ) генерируются через искусственный интеллект, поэтому качество зависит от модели.
- Если бот не смог собрать согласованную карточку, он не показывает потенциально неверную карточку, а предлагает попробовать следующую запись.
- Если в словаре совсем нет записей с переводом, тренажёр не запускается — бот предложит сначала открыть словарь и что-то туда добавить.
- Записи не повторяются, пока не закончится весь список; после этого список начинает крутиться заново.
- Если долго не отвечать или уйти из тренажёра, его состояние может устареть — тогда бот попросит открыть тренажёр заново.

## Что можно улучшить
- Сохранять отдельный статус `mastered` для кнопки `✅ Выучил`, если понадобится полноценная интервальная модель повторения.
- Чаще возвращать правила, по которым пользователь несколько раз ошибался.
- Добавить накопительную статистику по типам ошибок.
