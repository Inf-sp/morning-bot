# Тренажёр фраз

## Назначение
Тренажёр фраз учит применять устойчивые конструкции в новом контексте, а не дословно вспоминать исходную фразу из словаря. Пользователь сначала видит учебную карточку с фразой, переводом и коротким правилом, а затем решает отдельный тестовый пример по тому же правилу.

Главная идея: учебная фраза и тестовая фраза — разные сущности. Например, если карточка объясняет `ziek door iets`, тест должен проверять это правило на новой фразе вроде `Zij blijft thuis ____ de regen`, а не повторять `Hij is ziek ____ stress`.

## Как это работает
Пользователь заходит в раздел языка и нажимает «Тренажёр фраз». Бот берёт случайную фразу из личного словаря пользователя (только записи с переводом на русский) и генерирует для неё учебную карточку и отдельный тест.

**Учебная карточка.** Сначала бот показывает фразу целиком, перевод и короткое правило:

```
🧩 Фраза-тренажёр

Hij is ziek door stress.
Перевод: Он болен из-за стресса.

💡 ziek door iets
= болеть из-за чего-то

Дальше проверим это правило на новом примере.
```

Под карточкой две кнопки:
- `🧩 Тест` — открывает тестовый пример по правилу из карточки.
- `✅ Выучил` — не открывает тест, не засчитывает правильный ответ и сразу показывает следующую учебную карточку.

**Тест.** После нажатия `🧩 Тест` бот отправляет quiz poll в Telegram. На экране теста не повторяется учебная фраза. Показывается только новая фраза с пропуском:

```
🧩 Проверь себя

Zij blijft thuis ____ de regen.
```

Вариантов ответа четыре: правильный целевой токен и три правдоподобных неправильных варианта той же роли. Объяснение poll оформлено так же, как в тренажёре слов: полная фраза и русский перевод через стрелку.

Пример:

```
Zij blijft thuis door de regen. → Она остаётся дома из-за дождя.
```

**Если ответ правильный.** Бот показывает компактный результат без повторного большого объяснения:

```
✅ Верно

Zij blijft thuis door de regen.
Она остаётся дома из-за дождя.
```

Дальше доступна кнопка `Следующая фраза`.

**Если ответ неправильный впервые.** Бот показывает только короткое исправление:

```
❌ Правильный ответ: door

Zij blijft thuis door de regen.
Она остаётся дома из-за дождя.

💡 door = из-за, по причине чего-то
```

Дальше доступны кнопки `Новый пример` и `Дальше`. `Новый пример` генерирует другую тестовую фразу по тому же правилу, без повторного показа учебной карточки. `Дальше` открывает следующую учебную карточку.

**Если пользователь ошибся повторно по тому же правилу.** Бот не повторяет учебную карточку автоматически. Он показывает компактный результат `❌ Не закрепилось`, правильный ответ, полную тестовую фразу, перевод и короткую подсказку. В этом случае дополнительно появляется кнопка `🔎 Разобрать`.

`🔎 Разобрать` показывает короткий разбор только по явному нажатию. Разбор ограничен простыми словами, не повторяет полный учебный экран и не открывается автоматически.

## Связь учебной карточки и теста
Учебная фраза, правило и тест должны совпадать по смыслу. Перед показом карточки бот проверяет:
- русский перевод относится именно к учебной фразе;
- паттерн реально присутствует в учебной фразе;
- целевой токен используется в роли, которую описывает правило;
- пример естественный на выбранном языке;
- тестовая фраза проверяет то же правило;
- тестовая фраза не является копией учебной;
- в одной карточке не смешаны разные значения одного слова.

Пример запрещённой смешанной карточки:

```
Dat is bijzonder.
Перевод: Эта машина необычно дорогая.

💡 bijzonder + прилагательное
= необычно + прилагательное
```

Так делать нельзя, потому что `Dat is bijzonder` значит «Это необычно» или «Это особенное», а фраза «Эта машина необычно дорогая» должна быть `Deze auto is bijzonder duur`.

Правильное разбиение — две разные карточки:

```
Dat is bijzonder.
Перевод: Это необычно.

💡 bijzonder
= особенный, необычный
```

```
Deze auto is bijzonder duur.
Перевод: Эта машина необычно дорогая.

💡 bijzonder + прилагательное
= особенно, необычно + прилагательное
```

## Другие значения
Блок `Другие значения` показывается только если он коротко помогает и не конфликтует с главным правилом карточки. Формат — максимум одна строка:

```
Другие значения:
• bijzonder = особенный, необычный
```

Бот не показывает этот блок, если он повторяет главное правило, создаёт другое значение, не используемое в учебной фразе, или делает карточку длиннее без пользы.

## Основные возможности
- Отдельный тренажёр для нидерландских и английских фраз.
- Учебная карточка с фразой, переводом и коротким правилом.
- Тест на применение правила в новом контексте, а не на дословное восстановление исходной фразы.
- Четыре варианта ответа в Telegram poll.
- Компактный результат после правильного ответа без повторного объяснения.
- Короткое исправление после первой ошибки.
- Кнопка `🔎 Разобрать` после повторной ошибки по тому же правилу.
- Кнопка `Новый пример`, которая генерирует новую тестовую фразу по тому же правилу без повторного показа учебной карточки.
- Плавное чередование с тренажёром слов, если в словаре есть записи обоих типов.

## Настройки
- Выбор языка тренировки: нидерландский или английский — через соответствующий раздел меню.
- Если запустить тренажёр фраз напрямую (кнопка «Тренажёр фраз»), бот держится этого режима и не переключается самопроизвольно на тренажёр слов, даже если чередование в целом возможно.

## Ограничения
- Фраза попадает в тренажёр только если в словаре у неё заполнен перевод на русский; записи без перевода игнорируются.
- Слова и фразы — разные типы записей в словаре и тренируются раздельно; какая запись считается словом, а какая фразой, определяется автоматически по количеству значимых слов.
- Учебные карточки, тесты, варианты ответа и проверки согласованности генерируются через искусственный интеллект, поэтому качество зависит от модели.
- Если бот не смог собрать согласованную карточку или новый пример, он не показывает потенциально неверную карточку, а предлагает попробовать ещё раз или перейти дальше.
- Если фраз в словаре нет вовсе, тренажёр фраз тихо переключится на тренажёр слов (и наоборот, если нет слов — переключится на фразы).
- Если в словаре совсем нет ни слов, ни фраз с переводом, тренажёр не запускается — бот предложит сначала открыть словарь и что-то туда добавить.
- Фразы не повторяются, пока не закончится весь список фраз в словаре; после этого список начинает крутиться заново.
- Если долго не отвечать или уйти из тренажёра, его состояние может устареть — тогда бот попросит открыть тренажёр заново.

## Что можно улучшить
- Сохранять отдельный статус `mastered` для кнопки `✅ Выучил`, если понадобится полноценная интервальная модель повторения.
- Чаще возвращать правила, по которым пользователь несколько раз ошибался.
- Добавить накопительную статистику по типам ошибок: предлоги, порядок слов, устойчивые конструкции.
- Дать пользователю возможность вручную попросить ещё один пример даже после правильного ответа.

## Обновления
- Фразовый тренажёр использует учебную карточку, тест на новом примере, компактный результат и разбор только по запросу.
- Добавлена проверка согласованности учебной фразы, перевода, правила и тестового примера.
- Экран теста не повторяет учебную фразу, а правильный ответ не сопровождается длинным объяснением.
