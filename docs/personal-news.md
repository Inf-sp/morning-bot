# Новости для тебя

## Назначение

Раздел `📰 Новости для тебя` работает как персональный радар, а не как обычная лента. Он должен отвечать на вопрос: что изменилось рядом с пользователем или в его интересах, и почему это важно сейчас.

Основной профиль по умолчанию: Нидерланды, Алкмар. Поэтому локальные и нидерландские темы ищутся на нидерландском языке. Английский используется только для международных технологических тем: OpenAI, Apple, Telegram, Cloudflare, Railway, API и похожие сервисы.

## Где находится код

Основной модуль: `personal_news.py`.

Ключевые функции:
- `collect_personal_news(user_profile, sources=None, now=None, search_fn=None)` — основной pipeline отбора.
- `_queries_for_profile(profile)` — генерация коротких запросов по категориям.
- `_search_all_for_profile(profile)` — получение кандидатов через Tavily.
- `strict_filter(items, period="today", now=None)` — первичная фильтрация по дате, url, title и признаку конкретного изменения.
- `_score_candidate(item, profile, now=None)` — детерминированный скоринг 0-100.
- `_select_diverse(items)` — лимит 5 новостей, не больше одной новости из категории и не больше двух из большой группы.
- `_build_card(items, updated_ts=None, stale=False)` — компактный Telegram-текст и кнопки источников.
- `_save_history(cid, items, now=None)` и `_history_has_match(...)` — антидубли на 14 дней.

## Pipeline

1. Бот собирает профиль пользователя: город, страна, watchlist, любимые артисты и список сервисов, важных для работы бота.
2. Для каждой категории строится отдельный короткий запрос.
3. Tavily используется только как поисковый инструмент. Решение о показе принимает наш код.
4. Результаты нормализуются: url приводится к каноническому виду, дата публикации парсится, категория и язык берутся из запроса.
5. Статьи без даты публикации отбрасываются.
6. Статьи старше допустимого окна отбрасываются.
7. Отбрасываются записи без признака конкретного изменения: `wijziging`, `storing`, `staking`, `opening`, `recall`, `API update`, `pricing` и похожие маркеры.
8. Дубли удаляются по url, нормализованному title и похожести title.
9. История показов за последние 14 дней проверяется до скоринга, чтобы не повторять уже отправленную или очень похожую новость.
10. Каждая оставшаяся новость получает score от 0 до 100.
11. Показываются только новости со score >= 70.
12. Финальный список ограничивается пятью новостями с разнообразием категорий.
13. Показанные новости сохраняются в историю.
14. Telegram получает компактную карточку без debug-деталей.

## Языки поиска

Для Нидерландов, Алкмара, gemeente, NS, DUO, медицины, жилья, событий, еды, погоды и inburgering используется `nl`.

Для AI, OpenAI, Apple, Telegram, Cloudflare, Railway и API используется `en`.

Русский язык в поисковых запросах не используется для нидерландских новостей, потому что он хуже находит локальные источники и чаще приводит к нерелевантным пересказам.

## Категории

Поддерживаются категории:
- `city` — Алкмар и рядом: gemeente, дороги, события, новые места, безопасность.
- `north_holland` — новости Noord-Holland, которые могут затронуть Алкмар.
- `netherlands` — правила и изменения на уровне страны.
- `transport` — NS, OV, задержки, компенсации, расписания, забастовки.
- `housing_money` — аренда, huurtoeslag, налоги, ипотека, жильё.
- `documents_study` — DUO, inburgering, gemeente, экзамены, обучение.
- `health` — huisarts, аптеки, страховки, лекарства, zorg.
- `tech` — OpenAI, Telegram, Apple, Cloudflare, Railway, API.
- `leisure` — кино, сериалы, концерты, события рядом.
- `food` — AH, Jumbo, новые кафе и рестораны, food recalls.
- `wardrobe_weather` — сильный дождь, ветер, жара, UV и риски для велосипеда.
- `travel` — Schiphol, NS International, забастовки, правила поездок.
- `language` — нидерландский язык, экзамены, курсы, локальные языковые события.

## Источники и домены

Для Tavily запросов передаётся `include_domains`, когда для категории есть доверенный список. Это снижает шум и помогает локальным новостям.

Примеры доменов:
- Алкмар: `gemeentealkmaar.nl`, `alkmaarsdagblad.nl`, `nhnieuws.nl`, `streekstadcentraal.nl`, `noordhollandsdagblad.nl`, `indebuurt.nl`, `alkmaarcentraal.nl`.
- Нидерланды: `nos.nl`, `nu.nl`, `rtlnieuws.nl`, `ad.nl`, `telegraaf.nl`, `rijksoverheid.nl`.
- Транспорт: `ns.nl`, `9292.nl`.
- Документы: `duo.nl`, `inburgeren.nl`, `rijksoverheid.nl`.
- Медицина: `rivm.nl`, `ggd.nl`, `thuisarts.nl`, `apotheek.nl`, `zorgwijzer.nl`, `independer.nl`.
- Технологии: `openai.com`, `apple.com`, `telegram.org`, `cloudflare.com`, `ai.googleblog.com`, `theverge.com`, `techcrunch.com`, `arstechnica.com`, `railway.com`.

## Свежесть

Дата публикации обязательна. Если дата не найдена, статья не показывается.

Окна свежести:
- urgent и погода: максимум 24 часа;
- обычные новости: максимум 3 дня;
- локальные события и темы рядом: максимум 7 дней.

Новости старше 7 дней не показываются вообще. Старые материалы 2024/2025 годов не проходят эти окна, если у них нет новой даты публикации.

## Скоринг

Каждая новость получает score от 0 до 100.

Положительные факторы:
- `+30` — касается Алкмара или рядом.
- `+25` — влияет на деньги, документы, здоровье, транспорт, жильё, поездки, еду или практичные локальные планы.
- `+20` — связано с интересами пользователя: watchlist, артисты или сервисы вроде OpenAI/Telegram/Apple.
- `+15` — опубликовано в пределах свежего окна.
- `+10` — есть понятное действие или риск: проверить расписание, учесть забастовку, оформить, купить билет, проверить recall.

Штрафы:
- `-50` — дата публикации не определена.
- `-30` — статья старше допустимого окна.
- `-25` — слишком общая и без конкретного изменения.
- `-20` — нет личной пользы для пользователя.

Порог показа: `NEWS_MIN_RELEVANCE_SCORE = 70`.

## Антидубли

История хранится в kv-ключе `personal_news_history.json` минимум 14 дней.

Для каждой показанной новости сохраняется:
- `url`;
- `title`;
- `normalized_title`;
- `hash`;
- `source`;
- `category`;
- `date_sent`.

Новость не показывается повторно, если совпал url, совпал hash или title достаточно похож на уже показанный title. Похожесть считается по пересечению нормализованных слов.

## Разнообразие выпуска

Финальный выпуск:
- максимум 5 новостей;
- не больше 1 новости из одной категории;
- не больше 2 новостей из одной большой группы, например `netherlands`;
- приоритет сортировки: сначала score, затем порядок категорий с преимуществом Алкмара и Noord-Holland.

Если есть только 3 хорошие новости, бот покажет 3. Он не добирает выпуск слабым материалом.

## Формат сообщения

Карточка компактная:

```text
📰 Новости для тебя

📍 Алкмар
Gemeente Alkmaar meldt wegwerkzaamheden vandaag
Коротко: Gemeente Alkmaar meldt vandaag nieuwe wegwerkzaamheden en afsluiting.
💡 Почему важно: это рядом с Алкмаром.
Источник: Gemeente Alkmaar · сегодня
```

В карточке не выводятся:
- relevance score;
- raw query;
- search provider;
- debug info;
- служебное время обновления;
- формулировка `Почему тебе`.

Если хороших новостей нет, бот пишет:

```text
📰 Новости для тебя

Сегодня нет достаточно важных новостей для тебя.

Проверил:
• Алкмар
• Нидерланды
• NS / DUO
• AI / технологии
• досуг
```

## Кэш и расписание

Внешние точки входа не менялись:
- `send_period(bot, cid, period="today", force=False)`;
- `refresh(bot, cid, period="today")`;
- `send_scheduled(bot, cid)`.

Кэш выпуска остаётся в `personal_news_cache.json`. История показов отделена от кэша, поэтому устаревший кэш не используется как источник правды для антидублей.

Расписание уведомлений в `bot.py` и настройки рассылок не менялись.

## Проверка поведения

При изменениях раздела проверять вручную или коротким локальным сценарием:
- старая новость старше 7 дней не попадает в выдачу;
- новость без даты не попадает в выдачу;
- повторный url и похожий title не показываются;
- в выпуске не больше одной новости из категории;
- локальная новость Алкмара получает высокий приоритет;
- Dutch query используется для Alkmaar и Netherlands;
- English query используется для OpenAI/Apple/Telegram;
- пустое состояние честно сообщает, что важных новостей нет;
- формат остаётся компактным и без debug-информации.
