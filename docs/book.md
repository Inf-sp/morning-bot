# 📚 Книги для тебя

## Назначение
Раздел помогает найти следующую книгу для чтения, опираясь на то, что пользователь уже читал и что ему понравилось. Бот присылает готовую карточку с обложкой и подробным разбором книги, без необходимости искать что-то самостоятельно.

## Как это работает
Основа рекомендации — список «Мои книги»: книги, которые пользователь отметил как понравившиеся. Если этот список пуст, бот сначала попросит прислать несколько прочитанных и понравившихся книг — без этого не получится понять вкус.

Дальше рекомендацию формирует искусственный интеллект: он получает список любимых книг, список уже прочитанного/отклонённого/сохранённого, а также свежие данные из интернета о книжных новинках и рейтингах — и предлагает пять сильных книг под этот вкус, с описанием сюжета, объяснением, почему стоит читать, и цитатой. Для показа обложки бот отдельно обращается к внешней библиотечной базе (по названию книги ищется изображение обложки).

Кнопка «Заменить» показывает следующую книгу вместо текущей — без повторов того, что уже отмечено как прочитанное, любимое, отклонённое или уже показанное. Если ИИ временно недоступен или не смог предложить новую книгу, бот подстрахуется и предложит одну из проверенных классических книг, которую пользователь ещё не видел.

## Основные возможности
- Показ карточки с рекомендацией книги: обложка, автор и название (с оригинальным названием и годом, если есть), краткое описание, причины прочитать, краткий пересказ сюжета и цитата из книги.
- Кнопки под карточкой: «✨ Заменить» — широкая строка сверху, показывает другую книгу вместо текущей без повторов уже виденного; ниже, в два столбца — «❤️ В любимые» (явно переносит книгу в «Мои книги», влияет на вкус и подбор) и «⭐️ Сохранить» (кладёт книгу в список «Почитать», буфер, не влияет на вкус, и сразу показывает следующий вариант).
- Список «Почитать» — книги, отложенные для будущего чтения, с возможностью очистить список.
- Быстрый переход к настройкам книг прямо с карточки рекомендации.
- Сбор начального списка любимых книг, если он ещё пуст (можно пропустить).

## Настройки
В разделе «Настройки досуга» → «Книги» можно посмотреть и отредактировать список «Мои книги» — именно на эти книги ориентируется бот при подборе вкуса. Отклонённые, прочитанные и удалённые книги попадают в общую внутреннюю категорию «Не рекомендовать»; отдельных кнопок «Скрытое» и «Прочитано» нет. Жанровые предпочтения бот выводит сам из состава списка любимых книг.

## Ограничения
- Без хотя бы одной книги в списке «Мои книги» раздел не подберёт рекомендацию и предложит сначала прислать список прочитанного.
- Обложка появляется, только если внешняя библиотечная база нашла книгу по названию; если нет — карточка показывается без обложки.
- В отличие от кино, для книг нет проверки по внешнему рейтингу — предложенная ИИ книга показывается как есть, если только она не встречается в списках «уже видел».
- Список «не показывать снова» собирается из любимых, прочитанного, отклонённого и уже показанного — если пользователь очистит эти списки, рекомендации могут начать повторяться.

## Что можно улучшить
- Добавить фильтр по жанру или объёму книги (например, показывать только non-fiction или только короткие книги).
- Показывать, где книгу можно купить или скачать легально.
- Учитывать оценки книг от других читателей, а не только вкус ИИ.
- Дать возможность попросить книгу под конкретный запрос («что-то лёгкое перед сном», «нон-фикшн про историю») прямо из карточки.
