# 🎬 Подбор кино

## Назначение
Раздел подсказывает, что посмотреть — фильм или сериал — и работает как персональный
рекомендательный сервис. Рекомендации строятся преимущественно на данных TMDb
(Recommendations + Similar по вашим любимым проектам), а не на случайных подсказках ИИ.
Пользователю не нужно ничего искать самому: бот присылает готовую карточку с постером,
деталями и объяснением, почему это рекомендовано именно ему.

## Как это работает

### Анализ вкуса (на лету)
Перед каждой рекомендацией бот заново анализирует вкус пользователя по его спискам:
любимые фильмы и сериалы, «Посмотреть», просмотренные, отклонённые («Не нравится»),
недавно показанные. Каждый любимый проект резолвится в TMDb, и из его данных
собираются приоритеты: любимые жанры, страны, повторяющиеся режиссёры, преобладание
фильмов или сериалов, средний рейтинг, современные или старые проекты. Это **приоритеты, а
не жёсткие ограничения** — они влияют на порядок, но не запрещают другое. Поскольку профиль
пересобирается каждый раз, любое действие (добавил в любимое, отклонил, сохранил) сразу
меняет будущие рекомендации — вкус «обучается» без ручной настройки.

### Источник рекомендаций — TMDb
Для каждого любимого проекта бот берёт из TMDb списки **Recommendations** и **Similar**
(для фильмов и сериалов свои эндпоинты). Это основной источник новых названий. Кандидаты
со всех любимых объединяются в общий пул; если один и тот же проект рекомендован сразу от
нескольких любимых, он получает приоритет («часто рекомендуют вместе»). ИИ подключается
только как запасной вариант, если данных TMDb недостаточно (например, список любимых пуст
или сервис недоступен), а также для классификации настроения и объяснения причин.

### Фильтрация и порог рейтинга
Из выдачи всегда исключаются: уже просмотренное, любимое, сохранённое, отклонённое,
недавно показанное и дубликаты. Кандидаты с рейтингом TMDb ниже 7.0 отсекаются; если после
фильтрации ничего не осталось, порог постепенно снижается до 6.8, затем до 6.5.

### Ранжирование
Оставшиеся кандидаты сортируются по совпадению со вкусом: жанры, страны, повторяющиеся
режиссёры, преобладающий тип, базовый рейтинг TMDb и частота появления от разных любимых.
Первый в списке и показывается.

## Карточка
Карточка содержит: постер, название (и оригинальное), год, тип (фильм/сериал), жанры,
рейтинг TMDb, описание, и блок **«💡 Потому что вам понравился …»** — с указанием того
любимого проекта, от которого пришла рекомендация. Причина строится автоматически из данных
TMDb, без общих фраз.

Для сериалов дополнительно показываются: число сезонов и серий (📊), статус
(🎥 Продолжается / ✅ Завершено), дата следующей серии (⏳ Следующая серия — …) или
«⏳ Новый сезон ожидается», примерная длительность серии.

Для фильмов: длительность (⏱️), страна (🌍) и студия (🎬), если известна.

## Кнопки
Под карточкой:
- **⭐️ Сохранить** — кладёт проект в закладки и в список любимых (усиливает вкус), затем
  показывает следующую рекомендацию.
- **✨ Заменить** — помечает проект как «не нравится» (в чёрный список) и показывает другой
  вариант.
- **🎭 По жанру** — открывает меню выбора жанра (в два столбца) и подраздел «😊 По настроению».

### По жанру
Кнопки жанров располагаются в два столбца (Драма, Комедия, Ужасы, Фантастика, Детектив,
Триллер, Романтика, Боевик, Фэнтези, Криминал, Приключения, Вестерн, Документальный,
Семейный). После выбора жанра рекомендации строятся внутри него (TMDb Discover) с учётом
вкуса пользователя.

### По настроению
В меню «По жанру» есть раздел «😊 По настроению» с вариантами вроде «что-нибудь лёгкое»,
«хочу подумать», «атмосферное», «запутанное», «на один вечер» и т.д. Настроение
классифицируется ИИ в набор жанров TMDb (с детерминированным запасным вариантом), после
чего кандидаты ищутся через TMDb — сам подбор остаётся на данных TMDb.

## Предпочтения кино
В «Настройках досуга» есть экран «🎬 Предпочтения кино». Это **приоритеты, а не запреты** —
они влияют на подбор, но не блокируют другие рекомендации:
- Любимые жанры — отмечаются галочками (мультивыбор).
- Тип: фильмы / сериалы / без разницы.
- Сериалы: только завершённые / любые.
- Новинки: только новинки / любые годы.
- Минимальный рейтинг: 6.5 / 7.0 / 7.5 / 8.0 (стартовый порог, с тем же понижением при
  пустом результате).
- Страны — отмечаются галочками.

## Умное обучение вкуса
Каждое действие меняет профиль предпочтений, потому что он пересобирается на лету:
- «⭐ Сохранить» / «в любимое» усиливает вес похожих жанров, режиссёров, стран, тем.
- «👎 Не нравится» / «Заменить» снижает вероятность похожих проектов (чёрный список).
Если пользователь несколько раз сохраняет проекты одной тематики, бот постепенно понимает
вкус без ручной настройки.

## Ограничения
- Рекомендации строятся из TMDb; при пустом списке любимых бот сначала просит прислать
  несколько понравившихся названий, чтобы было от чего отталкиваться.
- Порог рейтинга TMDb — 7.0 с понижением до 6.8 и 6.5; если и на 6.5 ничего нет, бот
  предложит сменить жанр/настроение.
- Данные (детали, постеры, рейтинги) зависят от наличия проекта в TMDb; названия из списков
  резолвятся по поиску, поэтому редкие/неоднозначные названия могут не находиться.
- Списки «не показывать снова» собираются из любимых, просмотренных, отклонённых,
  сохранённых и недавно показанных — при их очистке рекомендации могут начать повторяться.
- Профиль вкуса не хранит отдельные «веса» — он выводится из актуальных списков при каждой
  рекомендации.

## Что можно улучшить
- Учитывать актёров, а не только режиссёров, в анализе вкуса.
- Добавить фильтр по конкретным ключевым словам/темам TMDb.
- Показывать сразу несколько вариантов на выбор, а не по одному.
